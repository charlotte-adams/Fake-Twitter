After the tests are complete, you may wish to restore your database to its initial state (as specified in twitter-schema.sql). To do so, follow these steps:

  1. Quit the server (if it is running)
  2. Open twitter-schema.sql in Sublime Text
  3. Make sure that the Build System is set to "SQLite"
    1. Open Tools menu
    2. Click "Build System"
    3. Select "SQLite"
  4. Press Ctrl + B to run the build system
  5. Your database has been reverted!
